# Voice-Guided UPI Assistant

An Android application designed to enable visually impaired and elderly users to perform digital payments independently using voice guidance and QR scanning.

## Features
- QR Code Scanning using CameraX and ML Kit  
- Voice Output using Text-to-Speech (TTS)  
- UPI Data Parsing (merchant name and amount extraction)  
- Accessibility-focused interaction (hands-free guidance)  

## How it Works
1. User scans a QR code  
2. Application detects and extracts UPI details  
3. System announces payment information via voice  
4. User proceeds with confirmation or enters amount  

## Tech Stack
- Kotlin (Android)  
- CameraX  
- Google ML Kit (Barcode Scanning)  
- Android Text-to-Speech (TTS)  

## Objective
To provide an inclusive and accessible digital payment experience by reducing dependency on visual interfaces and enabling voice-driven transactions.
